# Contributing

Thanks for wanting to help improve Blockbusters: Aeronautics!

## Reporting Bugs

Open an [Issue](../../issues/new?template=bug_report.md) and fill out the template. Include:
- What happened vs. what you expected
- Steps to reproduce
- Your crash log (from `.minecraft/logs/latest.log`)
- Modpack version

## Suggesting Mods

Open an [Issue](../../issues/new?template=feature_request.md) with:
- Mod name and Modrinth/CurseForge link
- Why it fits the pack's theme
- Any known conflicts

## Version Policy

- Releases are tagged (e.g., `v1.0.0`) and the `.mrpack` is attached to the GitHub Release
- The server will always match the latest release version
- Breaking changes (world resets, major mod removals) will be noted clearly in the changelog
